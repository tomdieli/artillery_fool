# artillery_fool

A clumsy tribute to the old 'artillery duel' game I played on the C64.

Written in C using the ncurses lib. One or more curses dev packages may need to be installed. In addition, the ncurses and math libs may need to be linked, depending on the environment.

For example:

`gcc -o artillery_fool artillery_fool.c -lncurses -lm`

Anyway, works on my desk:)
